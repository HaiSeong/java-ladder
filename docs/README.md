### 기능 요구사항 정리
- [x] 사람의 이름을 갖는다.
  - [x] 이름은 최소 1글자 최대 5글자다.
- [ ] 사다리의 가로라인은 겹칠수 없다. (나란히 있을 수 없다.)
- [ ] 참여할 사람의 이름은 쉼표로 구분해 받는다.
  - [ ] 사람은 최소 2명 참가해야한다.
- [ ] 사다리의 높이를 입력받는다.
  - [ ] 최대 높이는 1 이상이다.
- [ ] 실행 결과를 출력한다.
  - [ ] 사다리를 출력한다.
    - [ ] 사다리는 5칸이다.
    - [ ] 5칸 중 가장 오른쪽에 `|`를 높이만큼 출력한다.

  - [ ] 이름을 출력한다.
    - [ ] 이름이 4글자 이하인 경우, 오른쪽에 하나의 공백을 두고 우측 정렬해 출력한다.
    - [ ] 이름이 5글자인 경우, 공백없이 우측 정렬해 출력한다.
